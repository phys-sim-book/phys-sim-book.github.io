Detailed Description

template<typename T>
class muda::DeviceBuffer< T >

A std::vector like wrapper of cuda device memory, allows user to:

  • resize
  • reserve
  • clear
  • fill
  • shrink_to_fit
  • make view or subview from it
  • make a safe viewer from it
